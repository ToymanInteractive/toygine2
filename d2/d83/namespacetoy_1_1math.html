Compares two fixed-point values for approximate equality.

Same combined absolute and relative epsilon test as the floating-point overload: small values by absEpsilon, large values by relEpsilon scaled by max(|a|, |b|). Applicable to any toy::math::fixed_point type.

Template Parameters
TFixed-point type; must satisfy toy::math::fixed_point.
Parameters
aThe first value.
bThe second value.
absEpsilonThe maximum absolute difference treated as equal (default: 8× numeric_limits<T>::epsilon()).
relEpsilonThe maximum relative difference (default: 64× numeric_limits<T>::epsilon()).
Returns
true if a and b are considered equal under the chosen tolerances, false otherwise.
