Compares two single-precision floats for approximate equality.

This function uses a combined absolute and relative epsilon test. Small values are compared by absolute difference (absEpsilon), large values by relative difference (relEpsilon) scaled by max(|a|, |b|).

Parameters
aThe first value.
bThe second value.
absEpsilonThe maximum absolute difference treated as equal (default: 8× machine epsilon).
relEpsilonThe maximum relative difference (default: 64× machine epsilon).
Returns
true if a and b are considered equal under the chosen tolerances, false otherwise.
