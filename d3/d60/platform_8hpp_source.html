32 Identifies the target platform for compile-time and runtime platform detection. Values use hexadecimal identifiers
33 starting from \c 0x1000, with each platform family assigned a distinct range. This design enables compile-time
34 platform checks and potential combination with CPU architecture identifiers.

48 // Compile-time platform detection
