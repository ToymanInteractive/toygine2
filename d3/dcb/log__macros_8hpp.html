Detailed Description

Call-site macros that build toy::log::Metadata and dispatch through toy::log::Backend.

Defines LOG_TRACE, LOG_DEBUG, LOG_INFO, LOG_WARN, and LOG_ERROR. Each macro instantiates a static constexpr toy::log::Metadata at the call site, gates the call on LOG_MAX_LEVEL via if constexpr, and forwards the format pattern and arguments to toy::log::Backend::push().

Note
Included by core.hpp only; do not include this file directly.

Internal helper that expands a single LOG_* call.

Instantiates a static constexpr toy::log::Metadata when level passes the LOG_MAX_LEVEL gate, then forwards fmt and __VA_ARGS__ to toy::log::Backend::push(). On retro targets without dead-code elimination the discarded if constexpr branch may still emit the metadata; raise LOG_MAX_LEVEL or enable optimizations to avoid it.
