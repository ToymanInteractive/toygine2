Detailed Description

Mathematical functions, fixed-point arithmetic, and numeric utilities.

Includes the general math utilities from toy::math (see math/utils.hpp):

  • abs: Absolute value for signed integers, floating-point (float, double, long double), and toy::math::fixed; branch-free where applicable.
  • isEqual: Approximate equality for two floating-point values with configurable absolute and relative epsilon (default 8× and 64× machine epsilon); supports float, double, long double.
  • deg2rad, rad2deg: Angle conversion (degrees ↔ radians) for floating-point and toy::math::fixed; use toy::math::constants (e.g. constants::pi_v) for the conversion factor.

Also in this group: toy::math::fixed, toy::math::ValidFixedPointTypes, toy::math::fixed_point; mathematical constants in toy::math::constants (pi_v, e_v, etc.) for toy::math::fixed_point and standard floating-point types; toy::math::Vector2D, toy::math::real_t.

Key Features

  • Constexpr support where possible
  • Exception safety (noexcept operations)
  • Type safety through C++20 concepts
  • No dynamic allocation in core math types

Compares two floating-point values for approximate equality.

This function uses a combined absolute and relative epsilon test. Small values are compared by absolute difference (absEpsilon), large values by relative difference (relEpsilon) scaled by max(|a|, |b|). Supports float, double, and long double.

Parameters
aThe first value.
bThe second value.
absEpsilonThe maximum absolute difference treated as equal (default: 8× machine epsilon for T).
relEpsilonThe maximum relative difference (default: 64× machine epsilon for T).
Returns
true if a and b are considered equal under the chosen tolerances, false otherwise.
