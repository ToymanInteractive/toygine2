Detailed Description

macOS implementation of the toy::assertion crash handler and stack trace capture.

Registers signal handlers for SIGBUS, SIGFPE, SIGSEGV, and SIGUSR1. On signal delivery, captures a demangled stack trace via backtrace(3) and dladdr(3) and forwards it to the registered toy::assertion::StackWalkCallback.
