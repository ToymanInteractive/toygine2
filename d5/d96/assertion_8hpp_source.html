69/*!
70 \brief Callback function type for assertion failure handling.
71
72 This callback is invoked when an assertion fails. The callback receives a formatted assertion string containing the
73 assertion code, message (if any), file name, function name, and line number.
74
75 \param assertionString The formatted assertion string describing the failure.

195/*!
196 \brief Compile-time assertion function for constexpr contexts.
197
198 This function provides assertion capabilities in constexpr contexts where regular runtime assertions cannot be used.
199 When the \a condition is false, it throws a std::invalid_argument exception with the provided \a message, causing
200 compilation to fail if evaluated at compile time.
201
202 \param condition The condition to check. Must be true for the assertion to pass.
203 \param message The error message to include in the exception if the assertion fails.
204
205 \throws std::invalid_argument If \a condition is false, with \a message as the error description.
206
207 \pre The \a message pointer must point to a valid null-terminated string.
208
209 \note This function is designed for use in constexpr contexts where traditional assertions are not available.
210 \note Unlike regular assertions, this function always throws an exception in both debug and release builds when the
211 condition fails.
212 \note When evaluated at compile time, a failed assertion will cause a compilation error.
213 \note When evaluated at runtime, a failed assertion will throw an exception.
214
215 \section usage Usage Example
216
217 \code
218 #include "core/assertion.hpp"
219
220 template <size_t N>
221 constexpr size_t validateSize() {
222 toy::assertion::constexpr_assert(N > 0, "Size must be greater than zero");
223 return N;
224 }
225
226 constexpr auto size = validateSize<10>(); // OK
227 // constexpr auto invalid = validateSize<0>(); // Compilation error
228 \endcode
229
230 \sa assert
231 \sa assert_message
232 \sa assertion()
233*/
234constexpr void constexpr_assert(bool condition, const char * message) {
235 if (!condition)
236 throw std::invalid_argument(message ? message : "constexpr_assert failed");
237}
