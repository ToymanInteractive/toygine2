Detailed Description

template<typename StringType>
class toy::OStringStream< StringType >

Template output string stream class for building strings from various data types.

OStringStream is a lightweight, high-performance output stream class that provides a std::ostringstream-like interface while using a fixed-size or custom string storage type. It supports formatting and inserting various data types into a string buffer, including integers, floating-point numbers, booleans, pointers, and string-like objects.

Template Parameters
StringTypeThe type of the underlying string storage. Must satisfy the toy::StringLike concept.

Usage Example

#include "o_string_stream.hpp"
#include "fixed_string.hpp"
// Create a stream with FixedString storage
toy::OStringStream<toy::FixedString<64>> stream;
// Insert various types
stream << "Value: " << 42 << ", Pi: " << 3.14159;
stream << '!' << CStringView(" Result: ") << true;
// Get the result
const auto result = stream.str();
