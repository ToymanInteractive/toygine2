Detailed Description

Broken-down UTC calendar date and time with millisecond resolution.

A plain aggregate populated by toy::chrono::SystemClock::now(). All fields are zero on platforms without an RTC.

Key Features

  • Millisecond resolution: millisecond carries the sub-second component; sub-millisecond precision is truncated.
  • Stub-safe: all fields are zero-initialized on platforms without RTC.
  • Aggregate type: no user-declared constructors; supports designated initializers and constexpr use.

Usage Example

#include "core.hpp"
const toy::chrono::CalendarTime ct = toy::chrono::SystemClock::now();
// ct.year == 2026, ct.month == 5, ct.day == 19, ct.hour == 14, ...

  • Size: 10 bytes (no padding; alignment 2).

Safety Guarantees

  • Type safety: fixed-width integer fields prevent silent narrowing.
  • Exception safety: aggregate; all operations are noexcept.
  • Memory safety: no dynamic allocation; suitable for embedded and constrained targets.
