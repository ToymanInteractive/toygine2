size_t destSize,
size_t skipFrames = 1 )
noexcept

Captures the current call stack and formats it into dest.

Each frame is written as module!symbol + N bytes, one per line. C++ symbols are demangled via abi::__cxa_demangle when possible; raw mangled names are used on failure.

Parameters
destDestination buffer.
destSizeCapacity of dest in bytes.
skipFramesNumber of top frames to omit to hide signal-handling boilerplate (default: 1).

◆ handler()

void toy::assertion::anonymous_namespace{assertion_macos.cpp}::handler ( int signalId)
noexcept

Signal handler invoked on SIGBUS, SIGFPE, SIGSEGV, and SIGUSR1.

Formats the signal description and a demangled stack trace into a fixed-size buffer, then forwards the result to _stackWalkCallback. Terminates the process after the callback returns.

Parameters
signalIdPOSIX signal identifier delivered to the process.
