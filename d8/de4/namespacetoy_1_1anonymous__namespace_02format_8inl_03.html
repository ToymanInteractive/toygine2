Writes the argument at position index in the pack to stream.

Peels arguments one at a time via recursion, decrementing index at each step. When index reaches zero the current first argument is written. The recursion terminates either when the matching argument is found or when the pack is exhausted (which cannot happen on valid input because toy::FormatString validates index bounds at compile time).

Resolves the argument index for a single \{ ... \} placeholder body.

An empty body ({}) selects the next automatic index. Otherwise the body is read as a decimal numeral and returned as the zero-based argument index.

Template Parameters
PatternTypeType that provides at(size_t) for indexed access to the pattern's code units.
Parameters
patternFormat pattern string.
startInclusive index of the first code unit inside the opening brace.
endExclusive index of the closing brace; the examined range is [start, end).
autoIndexNext index to use for an empty placeholder; updated when start equals end.
Returns
Zero-based argument index to pass to the formatting dispatch.
Precondition
start <= end. Both indices refer to valid positions in pattern for the range [start, end). When start < end, code units in that range are decimal digits; this holds for patterns accepted by toy::FormatString.
Postcondition
If start == end, autoIndex is incremented by 1. Otherwise autoIndex is unchanged.
Note
Internal helper for formatTo() and vformatTo(); not part of the public API.
