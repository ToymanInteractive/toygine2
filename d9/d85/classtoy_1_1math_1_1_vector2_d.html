explicitconstexprnoexcept

Constructs a Vector2D from an array of values.

This constructor initializes a Vector2D with coordinates from the provided array. The array must contain at least two elements where values[0] is x and values[1] is y.

Parameters
valuesPointer to an array containing the x and y coordinates.
Precondition
The values pointer must not be null.
The array must contain at least two elements.
Postcondition
The vector is initialized with x = values[0] and y = values[1].

const real_t * toy::math::Vector2D::c_arr ( ) const
nodiscardconstexprnoexcept

Returns a constant pointer to the array representation of this vector.

This method provides read-only access to the internal array representation of the vector coordinates. The returned pointer points to the x coordinate, and the y coordinate follows immediately after.

Returns
A constant pointer to the array representation of this vector.
Note
The returned pointer points to the x coordinate.
The y coordinate can be accessed at pointer + 1.
The returned pointer is read-only and cannot modify the vector coordinates.
Use the non-const overload to allow modification.
