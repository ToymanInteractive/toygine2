class toy::math::fixed< BaseType, IntermediateType, FractionBits, EnableRounding >

Fixed-point numeric type with configurable storage and fractional precision.

Stores values as scaled integers using FractionBits fractional bits. Conversions from integral and floating-point types are explicit.

Template Parameters
BaseTypeStorage type for raw fixed-point value.
IntermediateTypeWider type used for intermediate calculations.
FractionBitsNumber of fractional bits.
EnableRoundingIf true, floating-point construction rounds to nearest value.
