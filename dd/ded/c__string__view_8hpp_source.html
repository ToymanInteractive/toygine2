30/*!
31 \class CStringView
32 \brief Non-owning string view class for C-style strings.
33
34 CStringView is a lightweight, non-owning wrapper around C-style strings that provides a safe and efficient way to work
35 with null-terminated character arrays. It offers a std::string_view-like interface while being specifically designed
36 for constexpr contexts and compile-time string manipulation.
37
38 Unlike std::string_view, CStringView is optimized for constexpr usage and provides additional functionality for
39 compile-time string operations. It does not own the underlying string data and simply holds a pointer to it, making
40 it extremely lightweight and efficient.
41
42 \section features Key Features
43
44 - ‚öôÔ∏è **Zero Allocation**: No dynamic memory allocation, only holds a pointer
45 - üîß **ConstExpr Support**: Most operations can be evaluated at compile time
46 - üõ°Ô∏è **Exception Safety**: All operations are noexcept
47 - üîó **STL Compatibility**: Provides std::string_view/std::string like interface
48 - üåç **UTF-8 Support**: Built-in UTF-8 character counting and manipulation
49 - üß¨ **Type Safety**: Uses C++20 concepts for type safety
50 - üìè **Non-owning**: Does not manage memory, safe for temporary strings
51
52 \section usage Usage Example
53
54 \code
