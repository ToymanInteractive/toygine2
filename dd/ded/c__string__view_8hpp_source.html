32/*!
33 \class CStringView
34 \ingroup String
35 \brief Non-owning string view class for C-style strings.
36
37 CStringView is a lightweight, non-owning wrapper around C-style strings that provides a safe and efficient way to work
38 with null-terminated character arrays. It offers a std::string_view-like interface while being specifically designed
39 for constexpr contexts and compile-time string manipulation.
40
41 Unlike std::string_view, CStringView is optimized for constexpr usage and provides additional functionality for
42 compile-time string operations. It does not own the underlying string data and simply holds a pointer to it, making
43 it extremely lightweight and efficient.
44
45 \section features Key Features
46
47 - ‚öôÔ∏è **Zero Allocation**: No dynamic memory allocation, only holds a pointer
48 - üîß **ConstExpr Support**: Most operations can be evaluated at compile time
49 - üõ°Ô∏è **Exception Safety**: All operations are noexcept
50 - üîó **STL Compatibility**: Provides std::string_view/std::string like interface
51 - üåç **UTF-8 Support**: Built-in UTF-8 character counting and manipulation
52 - üß¨ **Type Safety**: Uses C++20 concepts for type safety
53 - üìè **Non-owning**: Does not manage memory, safe for temporary strings
