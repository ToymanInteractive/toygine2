const Args &... args )
nodiscardconstexprnoexcept

Formats arguments into a new toy::FixedString using a compile-time validated pattern.

Iterates fmt, copies literal text verbatim, and substitutes each {} or {N} placeholder with the corresponding argument formatted via operator<< on toy::OStringStream.

Template Parameters
BufferSizeCapacity of the returned toy::FixedString in bytes.
ArgsTypes of format arguments; deduced from args.
Parameters
fmtCompile-time validated format pattern.
argsValues to substitute at each placeholder.
Returns
A toy::FixedString containing the formatted result.
Precondition
Each argument type supports operator<< on toy::OStringStream.
fmt is a valid toy::FormatString; validation occurs at compile time.
Note
{{ and }} in the pattern emit a literal { and } respectively.
When the formatted result exceeds BufferSize characters the output is silently truncated per toy::FixedString capacity semantics.

Formats arguments into output using a compile-time validated pattern.

Replaces the content of output with the formatted result of fmt applied to args. Literal text is copied verbatim; {} and {N} placeholders are replaced by the corresponding argument formatted via operator<< on toy::OStringStream.

Precondition
Each argument type supports operator<< on toy::OStringStream.
fmt is a valid toy::FormatString; validation occurs at compile time.
Postcondition
output holds the formatted result, subject to BackendType capacity/overflow semantics.

template<OStringStreamBackend BackendType, typename... Args>
void toy::vformatTo ( BackendType & output,
CStringView pattern,
const Args &... args )
noexcept

Formats arguments into output using a runtime pattern, type-erasing them automatically.

Type-erases each argument, then iterates pattern, copies literal text to output verbatim, and substitutes each {} or {N} placeholder with the corresponding type-erased argument. The pattern is validated at runtime; an invalid pattern triggers a debug assertion.

Template Parameters
BackendTypeType of the output string. Must satisfy the toy::OStringStreamBackend concept.
ArgsTypes of format arguments; deduced from args.
Parameters
outputDestination string; its previous content is replaced.
patternRuntime format pattern; may be a variable or computed string.
argsValues to substitute at each placeholder.
Precondition
pattern must be a valid format pattern consistent with sizeof...(Args) (same rules as toy::FormatString).
Each argument in args must remain valid for the duration of the call.
Postcondition
output holds the formatted result, subject to BackendType capacity semantics.
Note
{{ and }} in the pattern emit a literal { and } respectively.
Pattern validity is checked at runtime via assert_message with toy::validateFormatPattern(); use toy::formatTo() for compile-time checks.
See also
toy::vformat()
