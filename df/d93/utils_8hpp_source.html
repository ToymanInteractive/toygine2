33/*!
34 \brief Converts a Unicode UTF-8 encoded C string to a wide character string with character count limit.
35
36 This function translates a UTF-8 encoded C string into a wide character string stored in the destination buffer. The
37 conversion stops when the specified number of characters have been converted, the destination buffer is filled, or the
38 source string ends.
39
40 \param dest A pointer to the destination buffer where the converted wide character string will be stored.
41 \param destSize The size of the destination buffer in wide characters (not bytes).
42 \param src A pointer to the source UTF-8 encoded C string.
43 \param count The maximum number of characters to convert from the source string.
44
45 \return A pointer to the destination wide character string, or nullptr if the destination buffer is invalid.
46
47 \pre The destination buffer must be valid and have sufficient capacity.
48 \pre The source C string must be a valid UTF-8 encoded string.
49 \pre The count parameter must be reasonable (typically ≤ source string length).
50
51 \post The destination string is null-terminated.
52 \post The function returns nullptr on buffer overflow or invalid input.
53
54 \note Only BMP (≤ 0xFFFF) characters are supported by design; 4-byte UTF-8 sequences are not produced.
55 \note The function handles UTF-8 validation and skips invalid sequences.
56 \note If count exceeds the available characters, conversion stops at the end of the source string.
57*/
58wchar_t * utf8toWChar(wchar_t * dest, std::size_t destSize, const char * const src, std::size_t count) noexcept;

308 \brief Converts a 32-bit floating-point number to its string representation with specified precision.

363/*!
364 \brief Formats a number string by inserting grouping separators.
365
366 This function inserts a grouping separator (e.g., comma, space, or dot) into a number string every three digits,
367 starting from the right. This is commonly used for formatting large numbers to improve readability (e.g.,
368 "1,234,567").
369
370 \param buffer A pointer to the buffer where the formatted string is stored.
371 \param bufferSize The size of the buffer in characters.
372 \param separator A pointer to the grouping separator C string to insert (e.g., ",", " ", ".").
373
374 \pre The buffer must be valid and contain a null-terminated number string.
375 \pre The bufferSize must be sufficient to accommodate the formatted result.
376 \pre The separator must not be null and should be reasonable in length (≤ 8 characters).
377 \pre The input string should contain only digits (no decimal points, signs, or other characters).
378
379 \post The buffer contains the formatted number string with grouping separators.
380 \post The function modifies the buffer in-place.
381 \post The function returns early if the buffer is too small for the result.
382
383 \note The function validates available capacity before modification.
384 \note Grouping separators are inserted every three digits from the right.
385 \note The function handles edge cases gracefully (empty string, single digits, etc.).
386 \note The function is thread-safe for single-buffer operations.
387 \note Performance is optimized for common number string lengths.
388 \note The function does not validate that the input is purely numeric.
389*/
390void formatNumberString(char * buffer, std::size_t bufferSize, const char * separator) noexcept;
