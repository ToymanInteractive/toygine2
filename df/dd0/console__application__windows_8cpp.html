nodiscardnoexcept

Converts wide-character (UTF-16) command-line arguments and environment strings to UTF-8.

Allocates heap memory for the converted arrays. The caller is responsible for releasing all allocations via clearEntryPointArgs().

Parameters
argcNumber of command-line arguments.
argvwWide-character argument array of argc entries, or nullptr.
argvReceives a newly allocated null-terminated UTF-8 argument array, or nullptr if argvw is nullptr or argc is not positive.
envpwNull-terminated wide-character environment array, or nullptr.
envpReceives a newly allocated null-terminated UTF-8 environment array, or nullptr if envpw is nullptr.
Returns
true on success; false if any heap allocation fails.
Postcondition
On success, *argv and *envp each point to a null-terminated array of heap-allocated UTF-8 strings, or nullptr when the corresponding source array was nullptr or empty.
Warning
On failure, any partially allocated output must still be released via clearEntryPointArgs() to avoid leaks.
