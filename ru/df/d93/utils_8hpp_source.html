33/*!
34 \brief Converts a Unicode UTF-8 encoded string to a wide character string.
35
36 This function translates a UTF-8 encoded source string into a wide character string stored in the destination buffer.
37 The conversion stops when the specified number of characters have been converted or the destination buffer is filled.
38 The destination string is null-terminated.
39
40 \param dest A pointer to the destination buffer where the converted wide character string will be stored.
41 \param destSize The size of the destination buffer.
42 \param src A pointer to the source UTF-8 encoded string.
43 \param count The number of characters to convert from the source string.
44
45 \return A pointer to the destination wide character string, or nullptr if the destination buffer is invalid.
46*/
47wchar_t * utf8toWChar(wchar_t * dest, std::size_t destSize, char const * src, std::size_t count);

297/*!
298 \brief Format number string.
299
300 This function inserts a grouping separator (e.g., a comma or a space) into a number string. The separator is inserted
301 every three digits, starting from the right. The function assumes that the destination buffer is large enough to hold
302 the modified string.
303
304 \param buffer The destination buffer where the modified string is stored.
305 \param bufferSize The size of the destination buffer.
306 \param groupingSeparator The grouping separator to be inserted into the string.
307
