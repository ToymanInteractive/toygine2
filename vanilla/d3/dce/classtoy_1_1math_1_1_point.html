Detailed Description

2D integer point class for UI and input handling.

Point is a lightweight, high-performance 2D point class that uses integer coordinates. It is specifically designed for UI elements, mouse input handling, window positioning, and GUI layout calculations where pixel-perfect positioning is required.

The coordinate type is configurable through the value_type alias, allowing for different integer precisions based on application needs (defaults to std::int32_t).
