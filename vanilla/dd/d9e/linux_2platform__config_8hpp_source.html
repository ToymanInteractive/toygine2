48/*!
49 \def assert(expression)
50 \brief Debug assertion macro for runtime \a expression checking.
51
52 This macro provides runtime assertion checking in debug builds. It evaluates the given \a expression and triggers an
53 assertion failure if the \a expression is false. In release builds, this macro expands to nothing.
54
55 \param expression The boolean expression to evaluate. Must be convertible to bool.
56
57 \note This macro is only active in debug builds (_DEBUG defined).
58 \note In release builds, this macro expands to ((void)0) and has no effect.
59 \note Assertion failures will call toy::assertion::assertion with file, function, and line information.
60 \note The \a expression is evaluated only once and should not have side effects.
61
62 \warning Do not rely on the \a expression being evaluated in release builds.
63 \warning The \a expression should not have side effects as it may not be evaluated in release builds.
64
65 \see assert_message
66 \see toy::assertion::assertion
67*/
